NumPy库

  • NumPy 由 Travis Oliphant 于 2005 年创建， 指的是数值 Python（Numerical Python）。

  • NumPy 是用于处理数组的 python 库，它还拥有在线性代数、傅立叶变换和矩阵领域中工作的函数。

  • NumPy 旨在提供一个比传统 Python 列表快 50 倍的数组对象, 其中的数组对象称为 ndarray，它提供了许多支持函数，使得利用 ndarray 非常容易。
